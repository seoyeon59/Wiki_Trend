
# Wikipedia Correction Trend Dashboard

위키피디아 수정 데이터를 기반으로 트렌드를 분석하고 시각화하는 대시보드 프로젝트입니다.

기존 Figma Make 기반 UI 코드를 바탕으로, 프로젝트 목적에 맞게 대시보드 구조와 지표를 수정하고 있습니다.

## 프로젝트 목적

- 위키피디아 수정 데이터를 수집합니다.
- 수정량 및 관련 지표를 바탕으로 트렌드를 분석합니다.
- 대시보드 형태로 주요 변화를 시각적으로 확인할 수 있도록 합니다.

## 실행 방법

### 프론트엔드 실행
프로젝트에 필요한 도구를 설치합니다.

```bash
npm install

개발 서버를 실행합니다.

npm run dev
